# Cantor's Theorem
|A|<|P(A)|
---------------
PROOF:  
let g:A ↦ P(A) , g(a) = {a}  
⇒ (*) |A|≦|P(A)|  
//we'll continue to prove by contradiction  
Assume |A|=|P(A)|:  
⇒　∃ｆthat's a bijection from A to P(A)  
Define D = {a ∈ A:a ∉ f(a)}  
f is a bijection ⇒∃b∈A、ｆ（b）＝D  
There are two options:
- b∈D　⇒ｂ∉ｆ（ｂ）＝D　⇒ contradiction
- b∉D　⇒ｂ∈ｆ（ｂ）＝D　⇒ contradiction
⇒　|A|≠|P(A)|∧(*) ⇒　|A|＜|P(A)|
