# Cantor's Theorem
```|A|<|P(A)|```  
proof:  
g:A ↦ P(A) , g(a) = {a} ⇒ |A|≦|P(A)|  
Assume |A|=|P(A)|:  
∃ｆa bijection from A to P(A)  
D = {a ∈ A:a ∉ f(a)}  
f is a bijection ⇒∃b∈A、ｆ（b）＝D  
There are two options  
- b∈D　⇒ｂ∉ｆ（ｂ）＝D　⇒ contradiction  
- b∉D　⇒ｂ∈ｆ（ｂ）＝D　⇒ contradiction  
⇒　|A|≠|P(A)| ⇒　|A|＜|P(A)|
